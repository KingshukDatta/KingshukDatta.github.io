## 🛠️ Built With

- **HTML5** — semantic structure and content
- **CSS3** — custom properties, animations, grid layout, and responsive design
- **Vanilla JavaScript** — smooth scrolling, cursor glow, and active nav highlighting
- **Google Fonts** — Cormorant Garamond, DM Mono, Outfit

## 📝 Customization Guide

Want to update something? Here's where to look inside `portfolio.html`:

| What to change | Where to find it |
|---|---|
| Your name / tagline | Inside `<!-- HERO SECTION -->` comment block |
| About Me text | Inside `<!-- ABOUT SECTION -->` comment block |
| Skills | Inside `<!-- SKILLS SECTION -->` comment block |
| Projects & GitHub links | Inside `<!-- PROJECTS SECTION -->` comment block |
| Email address | Inside `<!-- CONTACT SECTION -->` comment block |
| Social links | Inside `<!-- CONTACT SECTION -->` comment block |
| Colors & theme | `:root` CSS variables at the top of `<style>` |
